Freeshell Index

Freeshells are shells that you can use for free to make your own ghost! They are provided by artists such as myself as a development aid for those who can't or don't want to draw, or if you just need a quick shell for something. They can be made in a variety of ways, so make sure you check the readmes to see what kind of expressions and other options are available!

All freeshells listed here are free to use for your own ghost projects, with the following terms.
You must:

  • Mention in your ghost's readme that you are using my freeshell. I can be credited as "Zichqec", with the following link: https://ukagaka.zichqec.com/
  • Keep the readme file that is included with the freeshell as-is.
You may:
  • Modify the shells, including recoloring, drawing new expressions, combining expressions, etc.
  • Modify the settings files in any way you need.
  • Use the shell to create a thumbnail image for your ghost, or a preview image for use on the ghost's release page, etc.
You must not:
  • Use these freeshells to create commercial works.
  • Distribute any version of these freeshells on their own.
  • Use these shells for works outside the realm of ukagaka.
  • Modify these shells to be discriminatory or political, or use it in a work that contains these elements.
  • Modify these shells to add nudity or sexual elements.
In the event that a particular freeshell specifies different terms in its readme file, that freeshell's specific permissions take precedence. I would appreciate if you ensure that you are using the latest version of the freeshell, in case the terms have updated.
You don't have to notify me if you use my freeshells, but if you do I would love to see what you've made!
