Features

  • Enhances your SSP experience
  • Ghosts can be calibrated for better results
  • Ghosts can be individually excluded from Spectre's effects if you prefer
  • Pairs well with Ghost Balloon

About

Spectre is a plugin which enhances your SSP experience! Download it today and see for yourself the incredible changes it provides!! If you don't like the effects, you can always toggle it off from its menu. It is recommended to use the balloon Ghost Balloon with this plugin.

For best results, make sure to calibrate the plugin individually for each ghost you use it with! Open the plugin's menu, and if the main menu has an option asking you to calibrate the ghost, make sure to click it. If that ghost's developer has provided a custom calibration, this step will be automatically completed when you open the plugin's menu with that ghost!
