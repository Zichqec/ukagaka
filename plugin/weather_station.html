About

Weather Station is a plugin which gives you a one-stop solution to get weather data for ghosts. You can search your location by city, zip code, etc., and once you've selected a location, weather data will be broadcast to any ghost that is set up to receive it.

You can view a simple weather display in the plugin's own menu, or ghosts may react to it if their developers have them set up to do so!

The plugin also provides air quality data, astronomy data (sunrise and sunset times, moon phase, etc.), and 3 days of forecast data.

Any number of ghosts can make use of the weather data, without needing to trust each individual one with your location.
