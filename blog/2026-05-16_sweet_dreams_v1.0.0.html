The ghost Sweet Dreams is now available! This ghost was first published as a part of Ghost Masquerade 8, and I also wrote an afterword for it.

This version is slightly different from the masquerade release, and has the Japanese translation removed per my agreement with あーるでぃーさん at the start of the project. I also added a few minor things, including a homeurl so that it can receive network updates.

I think I covered it pretty well in my afterword, but this project was a lot of fun! I learned a lot about Satori, localization, and more about common conventions used in Japanese ghosts. I also received so many kind comments from folks who enjoyed this premise a lot.

Masquerade is always a ton of fun! I'm looking forward to next year, too.
