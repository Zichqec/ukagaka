Hoard of Shinies has been updated to v1.0.6. There is no new dialogue this update, but instead a couple of handy new settings.

Changelog:

  • Added squidloon as a standard balloon.

  • While using squidloon, an option will appear in the settings to allow you to change the balloon color.

  • Add an option for reduced motion, which halves the speed of most of the animations (excluding talking and blinking).

  • Talk rate selection now includes an underline, to accommodate balloons without a visible anchor color.

  • General code cleanup.

  • Updated to Aosora v0.1.12.
