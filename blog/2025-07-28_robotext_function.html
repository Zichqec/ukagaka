Release of a new function: RoboText. This function takes a dialogue string and outputs it in chunks of 3 characters at a time, with various options to adjust it. It's intended to give text a more robotic feel.

I created this function for ATM Machine's v2.0.2 update, because I wanted the "normal" ATM text to not display instantly, but still feel distinctly different from the other dialogue. It's a bit fiddly, and it doesn't like to play nice with SakuraScript tags or AutoPause, but it is a neat effect if you can get it working properly. Maybe one day someone can find a way to improve on this.
