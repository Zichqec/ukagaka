Leaky Pipe is now available! It's a shell I made for the ghost Hydrate, as a part of 8/20 Hydrate Shell Jam 2025.

The shell features a pipe which clings to the top of the screen, and drips down water occasionally. The frequency of the dripping can be adjusted with dressups, and there are several color options for both the pipe and the liquid!

I've wanted to make this shell for a long time, and I'm pretty happy with how it came together. I've come to really appreciate Hydrate shells which stick to the top, since it keeps it up out of the way! This one in particular I think will be nice to shrink down to half size when I'm working with limited screen space. I hope you will enjoy it too!
