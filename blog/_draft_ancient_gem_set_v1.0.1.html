The Ancient Gem set has updated to v1.0.1! That includes the Ancient Gem freeshell, Ancient Gem balloon, and Ancient Gem calendar skin.

I originally made this set for Etc. Jam 2025, but at the time I wasn't feeling well and couldn't add everything I wanted to it. At the time, I said that I was hoping to push an update within a week or two. Well, I said that on March 19th, and this update is releasing on March TODOth, so I'd say I did a pretty great job 😌

Okay but on a more serious note, it's a relief to finally have this done! I knew I could get it done, I was just really struggling around the time that this set originally released, and then it got pushed to the side as other more important tasks came up. But with Etc. Jam 2026 pending, I figured that was a good deadline to finally kick me into finally completing it! Having it finished before I start on my next freeshell project would make me feel a lot better.

The thing is, I've actually had the arm poses for the freeshell roughed out for ages, but I never rendered them because I could tell the proportions were off, and I couldn't seem to adjust them the right way. I'm chalking that one up to brain fog, it really made everything so much harder. Now that I'm not dealing with brain fog anymore, I had no trouble adjusting it! I feel so much better about how it is now. If I'd pushed through and finished it back then, it would have been really janky.

Apparently I had also created collisions and the eye shine animation soon after the end of jam. I did it so soon after that I completely forgot they were not part of the original release! I thought those were already out there, but I've actually been sitting on them this whole time. Oops.

The opposite is true for the balloon, though. I thought that all the color variants of the balloon were already created and released, and then as I was preparing my update materials... I noticed that the balloon only had the main color. Oops!

Despite those hurdles, everything should finally be as I had originally intended it. All the parts I wanted for the shell exist, the balloon has all its colors, and I even fixed up a couple minor things in the calendar skin. I feel good about moving on to the next thing now.

Galla's Forged Metal set that complements this one has also just been updated, so you should check that out too! They're set up to slot together with minimal effort; if you want to combine them together you would only need to change the numbers of a few surfaces/balloons. Either way, be sure to check it out and see all the pretty colors Galla added.

Next up will most likely be some projects for Etc. Jam 2026, so look forward to that!
