Events

Sometimes I host ukagaka-related events! This index is to document them all in one place.

Note: This is for events that I personally host as an individual. This does not include events that I host for the Ukagaka Dream Team. I like to keep the two distinct, as these ones that I host myself are something I tend to run more casually.

If you'd like to see events that I've run as a moderator of the Ukagaka Dream Team, you can find them amongst other events on the Ukagaka Dream Team Wiki.

8/20 Hydrate Shell Jam

This is a short 1-day jam that takes place every year on August 20th, to celebrate "Hydrate day" by making a new shell for the ghost Hydrate. It's silly and creative and fun!

Aughost

This is a guided personal challenge event along the lines of Inktober, where I create 100 writing prompts, and participants are encouraged to do some writing for their ghost projects. How you approach it and how much you make is up to you!
