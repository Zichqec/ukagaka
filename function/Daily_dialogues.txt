//Ok! Here's a quick explanation of switch statements in YAYA. THESE ARE NOT EQUIVALENT TO SWITCH STATEMENTS IN OTHER LANGUAGES. Other languages use the labels switch/case for what YAYA calls case/when.
//Instead, switch statements in YAYA take an integer value, and they pick the dialogue in the switch statement that is at that index. So if I do this...
/*
switch weekday //weekday starts from 0 for some reason, so 0 is sunday, 6 is saturday
{
	"Sunday" //Position 0
	"Monday" //1
	"Tuesday" //2
	"Wednesday" //3
	"Thursday" //4
	"Friday" //5
	"Saturday" //6
}
*/
//Depending on the value of weekday, it will choose the appropriate day name! If you put a set of brackets in place of one of those dialogues, it will choose randomly from the dialogues in the brackets.
